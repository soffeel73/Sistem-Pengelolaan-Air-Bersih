# Smart Air Desa

Aplikasi manajemen air desa untuk pencatatan meteran, tagihan, dan laporan keuangan.

## Fitur Utama
- Pencatatan Meteran Air (Petugas)
- Manajemen Pelanggan
- Cetak Struk & Rekapitulasi
- Laporan Keuangan (Income/Expense)
- Dashboard Statistik

## Teknologi
- **Frontend**: Vue.js, Vite, Tailwind CSS
- **Backend**: PHP (Native API)
- **Database**: MySQL

## Setup Lokal
1. Clone repository:
   ```bash
   git clone https://github.com/soffeel73/smart-air-desa.git
   ```
2. Install dependencies:
   ```bash
   npm install
   composer install
   ```
3. Konfigurasi Database:
   - Copy `api/.env.example` ke `api/.env`
   - Sesuaikan kredensial database Anda.
4. Jalankan Dev Server:
   ```bash
   npm run dev
   ```
